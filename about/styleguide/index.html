Base Style

In order to reduce the amount of redundant code, this website uses Eric Meyer's Reset CSS which can be found here. A CSS reet file, whether yours or someone else's, helps bring browser inconsistencies to a baseline, making it much easier to work with your own CSS files.


To include the reset.css file once you have downloaded it is simple using the import statement:


@import url('reset.css');

The CSS Engine will first render the reset.css file before analyzing your own.

Typography

This is a Level 1 Heading

The main page header of this guide is an h1 element. Any header elements may include links, as depicted in the example.


h1 {
  font-size: 3.5em;
  font-family: "ff-meta-serif-web-pro",serif;
  font-style: normal;
  font-weight: 500;
  line-height: 1.25em;
}

Groupings

Paragraphs

This website employs no use of p elements. For semantic purposes, h4 and h6 have replaced them. Due to the fact this website is a text-heavy website, p elements do not make any visual hierarchical importance. Save for the Style Guide, the p tag should be discouraged in its use.

Horizontal rule

The hr element represents a paragraph-level thematic break, e.g. a scene change in a story, or a transition to another topic within a section of a reference book. There are two types of hr being used in this website. A large class is applied to thematic breaks between h1 elements and the main content of the page.


hr {
  display: block;
  height: 1px;
  border: 0;
  border-top: 1px solid;
  rgb(204, 204, 204);
  margin: 1em 0;
  padding: 0;
}

hr.large {
  border: none;
  height: 10px;
  background-color: rgb(235, 235, 235);
  background-color: rgba(255, 255, 255, 0.2);
  margin: 20px 0 50px 0;
}

Figures

Figures are usually used to refer to images. Figures in this website, with the exception of the Musings Page, adhere to an img link. Images will take the user to a predefined contextual or complementary location, internally or externally. It is important to note the feedback present within the img hover links. These provide for feedback, alerting the user that the images are able to be clicked on.





#iconsList a img {
  opacity:1;
  filter:alpha(opacity=100);
  -webkit-transition: opacity 0.3s ease;
  -moz-transition: opacity 0.3s ease;
  -o-transition: opacity 0.3s ease;
  transition: opacity 0.3s ease;
}

#iconsList a img:hover{
  opacity:0.3;
}
