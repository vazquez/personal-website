I Am Robot: Responsive Blog Template

View Project
website

Challenge

People, like, live on their cell phone. They are ttyl'ing and c u l8er'ing all the time. Those cool kids. Well, we have to get with the times! What if your audience was on mobile all day? What if you needed to deliver content? What if you wanted a simple solution?

Solution

This is a blog site template that is responsive and responsibly semantic. It has been a lot of fun creating this. This is a template, which can be applied as a Wordpress or Drupal theme, that aesthetically aims to achieve something that is modern, yet can withstand the test of time. I wanted something that won't look old fashioned in the near future. The blog is for robots, after all.
