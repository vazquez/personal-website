Private Media Ownership

View Project
print

Challenge

Information, too much information! I was commissioned by Openmedia.ca to take a chart of media ownership that the Canadian Media Guild developed and create a more graphically interesting version of it.

Solution

Alas, data visualization! Taking the idea of the primitive colour wheel, I implemented colours to connote media ownership percentages. At the bottom (seen top) I added what each network owns. By creating a rainbow effect, the viewer found it a friendly challenge to search for the colour that related to the network in the colour wheel. The end goal was informing a user, and them not even realizing it. Ninja, baby!

Fun Fact: I learned Excel thanks to this project.
